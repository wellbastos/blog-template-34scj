# Blog Template

JavaScript Bootstrap Blog Template with examples of Module Pattern, ES6 Classes and Export/Import, WebPack, Babel and Fetch Requests API and updates DOM.

## Setup

```
npm install
npm run start
```

## WebPack

```
npm install -g webpack-cli
npm run watch
```
